**Resume Builder**

**To Configure:**<br>
git clone https://github.com/bmkamath2000/ResumeBuilder.git<br>
cd ResumeBuilder/<br>
sudo apt update<br>
sudo apt upgrade<br>
sudo apt install python3-pip<br>
sudo apt install python3-django<br>
sudo apt install python3.12-venv<br>
python3 -m venv myvenv<br>
source myvenv/bin/activate<br>
pip install django-cors-headers<br>
pip install pkgconfig<br>
sudo apt install pkg-config<br>
sudo apt install cmake<br>
sudo apt install libcairo2-dev<br>
pip install xhtml2pdf<br>
pip install pdfkit<br>
cd ResumeBuilder/jumpwhere-master/<br>

**To Run:**<br>
python3 manage.py runserver<br>


After running the server you can head over to http://127.0.0.1:8000/<br>
